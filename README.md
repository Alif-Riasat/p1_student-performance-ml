# 🎓 Student Performance Prediction

## 📌 Overview

This project predicts whether a student will **pass or fail** using a Random Forest model based on academic and personal data.

---

## ⚙️ Tech Stack

* Python
* Pandas
* Scikit-learn
* Matplotlib

---

## 🎯 Target

* `pass` (1 = Pass, 0 = Fail)
* Created from final grade `G3`

---

## 🔄 Steps

* Load and clean dataset
* Convert grades to Pass/Fail
* Train Random Forest model
* Evaluate using confusion matrix

---

## 📊 Result

The model can predict student performance with good accuracy and shows how factors like study time and absences affect results.
